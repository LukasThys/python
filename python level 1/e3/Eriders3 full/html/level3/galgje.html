Dit is een vrij simpel spel. Het enige wat we nodig hebben is:

  • een div met id "contentGalgje"
  • een stukje tekst over hoeveel keer de speler nog mag raden
  • in dat stukje tekst een "strong" tag met id "aantalFout"
  • een knop (link!!) met id "knopLetter"
  • een knop (link!!) met id "knopWoord"

De tekst op de knoppen mag je zelf kiezen, wij stellen voor : "Raad een letter!" en "raad het woord!".
